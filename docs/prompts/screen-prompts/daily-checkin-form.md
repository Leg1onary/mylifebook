# Screen Prompt — DailyCheckinPage

## Файлы
- `frontend/src/pages/daily/DailyCheckinPage.tsx`
- `frontend/src/pages/daily/DailyHistoryPage.tsx`

## Контекст
Ежедневный чекин — основа регулярности. Занимает 1–2 минуты.
Открывается из TodayPage или через + modal.
Если чекин за сегодня уже есть — форма открывается в режиме редактирования (PUT).

## DailyCheckinPage — что реализовать

**Форма (React Hook Form + Zod):**

### Шкалы 1–10 (slider или tap-to-select)
Каждая шкала: label + визуальный индикатор текущего значения
- Настроение (mood)
- Энергия (energy)
- Тревога (anxiety)
- Стыд (shame)
- Одиночество (loneliness)
- Злость (anger)

UX: тап на цифру 1–10 или горизонтальный слайдер. На мобиле — крупные тапабельные кнопки.

### Emotion tags (мультивыбор)
Предустановленный список: тревога, стыд, грусть, злость, усталость, одиночество,
облегчение, радость, спокойствие, гордость, раздражение, пустота, вина, нежность
Стиль: pill-кнопки, при выборе — акцентный цвет

### Текстовые поля (все опциональные, textarea)
- «Что сегодня главное?»
- «Что сейчас болит?»
- «Что поддерживало сегодня?»
- «Что мне сегодня нужно?»

### Toggle
- «Сегодня был триггер» (bool)

### Кнопки
- [Сохранить] — POST или PUT в зависимости от режима
- [Отмена] — вернуться назад без сохранения

**Автосохранение черновика:**
- draftStore.setDraft('daily-checkin', formValues) при onChange
- debounce 1000ms
- При повторном открытии — восстановить черновик с вопросом «Продолжить незаконченный?»

**После сохранения:**
- Invalidate query ['today']
- navigate('/today')

## DailyHistoryPage — что реализовать

- Список чекинов: дата + mood индикатор + emotion tags preview
- Клик на строку → inline expand с полными данными
- Пустой стейт: «Пока нет записей. Начни с сегодняшнего дня.» + CTA

## API
```
POST /api/v1/daily-checkins       — создать
PUT  /api/v1/daily-checkins/:id   — обновить
GET  /api/v1/daily-checkins       — список для истории
```

## Валидация
- Все числовые поля: int 1–10 (если заполнены)
- date: автоматически сегодня, не показывать пользователю

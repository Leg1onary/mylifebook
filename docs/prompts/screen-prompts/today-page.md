# Screen Prompt — TodayPage

## Файл
`frontend/src/pages/today/TodayPage.tsx`

## Контекст
Главный экран приложения. Открывается после логина и при нажатии на «Today» в bottom nav.
Показывает состояние текущего дня и быстрый доступ ко всем действиям.

## Что реализовать

**Данные:** один запрос `GET /api/v1/today`

**Блоки экрана (сверху вниз):**

### 1. Хедер дня
- Дата сегодня: «Понедельник, 15 июня»
- Streak indicator: «🔥 7 дней подряд» (если streak_days > 1)

### 2. Статус чекина
- Если `checkin_done: false`:
  - Карточка с CTA: «Как ты сегодня?» + кнопка [Отметить день] → /daily/new
  - Мягкий акцент, не агрессивный
- Если `checkin_done: true`:
  - Карточка с мини-сводкой шкал (mood, anxiety, shame в виде цветных индикаторов)
  - Ссылка [Изменить]

### 3. Текущий фокус
- Если `current_week_focus` не null:
  - Блок «Фокус недели» с текстом фокуса
  - Серая карточка, небольшой шрифт

### 4. Активный старый закон
- Если `active_old_law` не null:
  - Строка «Следи за: "[old_law]"»
  - Subtle стиль, не доминирующий

### 5. Быстрые действия (grid 2x2 + 1)
- [Записать триггер] → открывает + modal или /triggers/new
- [Разобрать мысль] → /thought-records/new
- [Эксперимент] → /experiments/new
- [Свободная заметка] → /journal/notes/new
- [SOS 🚨] → отдельная кнопка, полная ширина, красный акцент → /sos

### 6. Счётчики активности
- «Незавершённых TR: N» (если > 0) с кнопкой [Посмотреть]
- «Активных экспериментов: N» (если > 0)

## Состояния
- loading: skeleton layout (хедер + карточка + кнопки)
- error: banner «Не удалось загрузить данные» + [Обновить]
- Нет данных за неделю: скрыть блоки фокуса и закона

## API
```
GET /api/v1/today
```

## Зависимости
- `frontend/src/api/` — запрос today
- TanStack Query: `useQuery(['today'])`
- Навигация через React Router `useNavigate`

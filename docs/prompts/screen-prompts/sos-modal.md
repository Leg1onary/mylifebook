# Screen Prompt — SOS Modal

## Файл
`frontend/src/components/sos/SOSModal.tsx` (или `/pages/sos/SOSPage.tsx`)

## Контекст
Открывается из + modal как overlay поверх текущего экрана.
Цель: зафиксировать острый момент за 60–90 секунд.
4 шага, по одному на экран. Прогресс-бар сверху.

## Что реализовать

**Контейнер:**
- Full-screen modal overlay (backdrop-blur + затемнение)
- Кнопка × для закрытия (с подтверждением если есть незаполненные данные)
- Swipe-down для закрытия на мобиле
- Прогресс-бар: 4 шага

**Шаг 1 — Ситуация**
- Заголовок: «Что происходит?»
- Поле: «Что случилось?» (textarea, autofocus, large)
- Поле: «Что я сейчас думаю?» (textarea)
- Кнопка [Далее →]

**Шаг 2 — Эмоции**
- Заголовок: «Что чувствуешь?»
- Emotion tags мультивыбор (крупные, тапабельные)
- Один общий слайдер интенсивности 0–10
- Кнопки [← Назад] [Далее →]

**Шаг 3 — Старый закон**
- Заголовок: «Какой старый закон включился?»
- Текст старого закона пользователя из personal_context (если есть) — показать как suggestion pill
- Textarea для ввода / подтверждения
- Стоп-фраза: если в профиле есть growth_goals — показать одну релевантную как «Помни:»
- Кнопки [← Назад] [Далее →]

**Шаг 4 — Заземление**
- Заголовок: «Один маленький шаг»
- Grounding suggestion (одна из заготовленных фраз):
  «Сделай 3 медленных вдоха»
  «Назови 5 вещей которые видишь»
  «Выпей стакан воды»
  «Напиши одно честное предложение о том что происходит»
- Кнопка [Я зафиксировал — сохранить]

**После сохранения:**
- Сохранить как TriggerEvent (POST /api/v1/triggers) со статусом черновика
- Показать brief confirmation (2 секунды): «Зафиксировано»
- Предложить: [Разобрать подробно] | [Закрыть]
- [Разобрать подробно] → /thought-records/new?trigger_id=:id

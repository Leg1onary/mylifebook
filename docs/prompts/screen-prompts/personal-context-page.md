# Screen Prompt — PersonalContextPage

## Файл
`frontend/src/pages/profile/PersonalContextPage.tsx`

## Контекст
Личный профиль пользователя — основа AI-персонализации.
Доступен из [Me] таба → Settings или из онбординг-баннера на TodayPage.
Два режима: просмотр/редактирование профиля и загрузка raw text.

## Что реализовать

**Layout:** одна страница с двумя секциями (tabs или scroll)

---

### Секция 1 — «Мой профиль»

**Если профиль пустой:**
- Empty state: «Профиль не заполнен. Добавь личный контекст, чтобы AI понимал тебя лучше.»
- Кнопка [Загрузить контекст] → scroll или switch на секцию 2

**Если профиль заполнен (read mode):**
- Список полей с возможностью редактировать (inline или [Редактировать] переключает в edit mode)

**Поля профиля:**

| Поле | Тип | Описание |
|---|---|---|
| triggers | textarea (каждый с новой строки) | Ключевые ситуации-триггеры |
| old_laws | textarea (каждый с новой строки) | Старые законы / схемы |
| typical_distortions | chips мультивыбор | Типичные искажения |
| communication_prefs | textarea | Как предпочитает общение и обратную связь |
| growth_goals | textarea (каждый с новой строки) | Цели роста |
| context_notes | textarea | Дополнительно |

- Кнопка [Сохранить профиль] (PUT /personal-context)
- Дата последнего обновления внизу

---

### Секция 2 — «Загрузить контекст»

- Заголовок: «Вставь текст о себе»
- Подсказка: «Можно вставить ответ из внешнего чата, дневниковую запись или любой текст о твоих паттернах. AI извлечёт из него структурированный профиль.»
- Textarea большой (min-height: 200px), placeholder: «Вставь текст здесь...»
- Toggle: «Заменить текущий профиль / Объединить с текущим» (merge: false/true)
- Кнопка [Извлечь профиль]
  - loading: спиннер + текст «AI извлекает профиль...»
  - success: показать extracted данные для просмотра перед сохранением
  - Кнопки [Сохранить] | [Изменить вручную]
  - error: «AI не смог обработать текст. Попробуй сократить или заполни профиль вручную.»

---

### Секция 3 — «AI статус» (внизу страницы)

- Toggle: AI включён / выключен
- Текст: «Когда AI включён, твои записи (без имён и идентификаторов) отправляются в OpenRouter для анализа.»
- Ссылка: «Что именно отправляется?» → раскрывающийся список полей

## API
```
GET  /api/v1/personal-context              — получить профиль
PUT  /api/v1/personal-context              — обновить вручную
POST /api/v1/personal-context/extract      — извлечь из raw text
GET  /api/v1/settings                      — для ai_enabled toggle
PUT  /api/v1/settings                      — обновить ai_enabled
```
